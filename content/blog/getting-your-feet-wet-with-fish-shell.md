## Software/Tool Versions {#software-tool-versions}

| Software   | Version |

| OSX        | 10.12.6 |

| Fish       | 2.7.0   |
| Oh My Fish | 6       |


## Installation {#installation}

```shell-script
brew install fish
```

Now, install oh-my-fish via `git` because curl-bash is for suckers!

```shell-script
git clone https://github.com/oh-my-fish/oh-my-fish
cd oh-my-fish
bin/install --offline
```
